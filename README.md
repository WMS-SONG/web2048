# 2048 🎮

一个功能丰富的 2048 网页游戏，支持多难度模式、暗夜主题、音效音乐、撤销操作、历史记录等特性。

## ✨ 特性

### 🎯 三种难度模式
| 模式 | 棋盘 | 目标分数 |
|------|------|----------|
| **简单** | 3×3 | 256 |
| **普通** | 4×4 | 2048 |
| **困难** | 5×5 | 4096 |

切换难度时自动保存当前进度，防止误操作丢失分数。

### 🌓 双主题切换
- **白天主题** — 经典暖色调
- **暗夜主题** — 深蓝冷色调，夜间护眼
- 棋盘在暗夜模式下带有微光效果

### 🎵 音效与音乐
- **Web Audio API** 合成音效（无需音频文件）
- 移动、合并、游戏结束均有不同音效
- 支持背景音乐播放（`bgm.mp3`）
- 音效和音乐独立控制

### ↩️ 撤销功能
- 每局游戏最多撤销 **3 次**
- 撤销前弹出确认窗口，防止误触
- 撤销后恢复上一步的棋盘、分数和步数

### 📊 历史记录
- 每个模式独立记录最近 **10 局** 成绩
- 显示分数、达成时间、是否为最佳成绩
- 超过 30 天的记录自动清理

### 🏆 最佳成绩
- 每个模式独立保存最高分
- 切换难度时自动切换显示当前模式最佳分

### 📱 移动端适配
- 触摸滑动操作
- 键盘方向键 + WASD 支持
- 触屏设备禁用页面滚动
- 底部键盘操作提示（移动端）

### 🎨 视觉特效
- 方块合并融合动画
- 分数浮动弹出效果
- 新纪录脉冲高亮
- 步数计数器

## 🚀 快速开始

直接打开 `index.html` 即可游玩，无需任何构建工具或依赖。

```
git clone https://github.com/WMS-SONG/web2048.git
cd web2048
start index.html
```

## 🕹️ 操作方式

| 操作 | 桌面端 | 移动端 |
|------|--------|--------|
| 移动方块 | 方向键 / W A S D | 滑动屏幕 |
| 开启/关闭音乐 | 点击音乐按钮 🎵 | 同左 |
| 切换主题 | 点击主题按钮 🌙/☀️ | 同左 |
| 撤销 | 点击撤销按钮 ↩️ | 同左 |
| 查看历史 | 点击历史按钮 📋 | 同左 |
| 新游戏 | 点击"新游戏"按钮 | 同左 |

## 🛠️ 技术栈

- **纯前端** — 单 HTML 文件，零依赖
- **CSS 自定义属性** — 主题切换系统
- **Web Audio API** — 游戏音效合成
- **localStorage** — 本地数据持久化
- **CSS 动画** — 融合特效、分数弹出
- **响应式设计** — 适配桌面和移动端
